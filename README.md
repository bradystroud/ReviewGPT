# ChatGPT PR Reviewer
### Overview 🌐
This project is a PR (Pull Request) reviewer built using .NET 7 and C#. It uses ChatGPT via the OpenAI API to automatically review and comment on PRs.

### Requirements 📦
- .NET 7

### Run 🛠
1. `dotnet run` 🚀
